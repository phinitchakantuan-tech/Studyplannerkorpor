อนุกรม
อนุกรมเลขคณิต: aₙ = a₁ + (n-1)d
d = ผลต่างร่วม, Sₙ = n/2(a₁+aₙ)
✏️ 1,4,7,10... → d=3
อนุกรม
อนุกรมเรขาคณิต: aₙ = a₁·rⁿ⁻¹
r = อัตราส่วนร่วม
✏️ 2,6,18,54... → r=3
ร้อยละ
กำไร% = (กำไร/ทุน) × 100
ราคาขาย = ทุน + กำไร
✏️ ทุน 100, ขาย 120 → กำไร 20%
ร้อยละ
ส่วนลด = ราคาปกติ × อัตราส่วนลด%
ราคาจริง = ราคาปกติ - ส่วนลด
✏️ 500 ลด 20% → จ่าย 400
ดอกเบี้ย
ดอกเบี้ยอย่างง่าย = PRT/100
P=เงินต้น, R=อัตรา%, T=เวลา(ปี)
✏️ 1000×5%×2 = 100 บาท
ดอกเบี้ย
ดอกเบี้ยทบต้น = P(1+r)ⁿ - P
r=อัตรา(ทศนิยม), n=จำนวนงวด
✏️ 1000(1.05)² - 1000 = 102.5
สมการ
โจทย์อายุ: กำหนด x=อายุปัจจุบัน
อนาคต +n ปี / อดีต -n ปี
✏️ ปีนี้ x, 5 ปีหน้า = x+5
สถิติ
Mean = ผลรวมทั้งหมด / จำนวน
Median = ค่ากลาง (เรียงลำดับก่อน)
✏️ Mode = ค่าที่ปรากฏบ่อยสุด
พื้นที่
สามเหลี่ยม = ½ × ฐาน × สูง
สี่เหลี่ยมผืนผ้า = กว้าง × ยาว
✏️ วงกลม = πr²
ปริมาตร
ทรงกระบอก = πr²h
ทรงกลม = (4/3)πr³
✏️ ทรงสี่เหลี่ยม = กว้าง×ยาว×สูง
โครงสร้างราชการ
ส่วนกลาง → ภูมิภาค → ท้องถิ่น
กระทรวง → กรม → จังหวัด → อำเภอ → ตำบล

คณิตศาสตร์
เทคนิคทำโจทย์เร็ว
  • อนุกรม → ดูผลต่าง 2-3 ตัวแรกก่อน
  • ร้อยละ → แทนค่า 100 แล้วคิดต่อ
  • โจทย์อายุ → ตั้ง x = อายุปัจจุบัน เสมอ
  • ตาราง → อ่าน หัวตาราง ก่อน แล้วค่อยดูโจทย์
  • ไม่แน่ใจ → ตัดตัวเลือกที่ผิดชัดออก แล้วเดา
เงื่อนไขสัญลักษณ์
วิธีแก้เงื่อนไขสัญลักษณ์
  • อ่านเงื่อนไขให้ครบก่อน อย่าเพิ่งตัดสิน
  • วาดตาราง 2×2 หรือ Venn diagram ช่วย
  • เงื่อนไข "ถ้า A แล้ว B" ≠ "ถ้า B แล้ว A"
  • ระวัง "บางส่วน" vs "ทั้งหมด"
  • ฝึกทำ 20 ข้อ/วัน จับเวลาด้วย
กฎหมาย
จำกฎหมายให้ได้คะแนน
  • จำ ชื่อกฎหมาย + ปี พ.ศ. ที่ออกบ่อย
  • โครงสร้างราชการ → วาด Mind Map ไว้
  • ทำ 200 ข้อ พี่แมงปอ → จด keyword ที่ผิด
  • จริยธรรม 7 ข้อ ท่องให้ขึ้นใจ
  • อุทธรณ์ 15 วัน / ฟ้องศาล 90 วัน
ภาษาอังกฤษ
เทคนิคทำข้อสอบภาษาอังกฤษ
  • Grammar → ดู Subject ก่อน แล้วเลือก Verb
  • Reading → อ่าน Question ก่อน Passage
  • Vocab → ดู context รอบๆ คำ ช่วยเดาความหมาย
  • บทสนทนา → จำ สถานการณ์ + response ที่เหมาะสม
  • ท่องผลึก 30 คำ ทุกวัน → ฟังเสียง + ความหมาย
